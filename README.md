## Development

Requires Node.js to be installed.

### Setup

```bash
yarn install
```

### To run locally:

* In a terminal/command line navigate to the project directory.
* Type `yarn install`
* Type `yarn start`
* Open a browser to `localhost:3000`

### To run tests:

* In a terminal/command line navigate to the project directory.
* Type `yarninstall`
* Type `yarn test`
